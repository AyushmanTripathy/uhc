the useful html compiler
------------------------

options
  -h : help
  -g : generate uhc.config.json
  -c : load custom config
  -w : watch dir

commands
  dev : watch src dir, recompile on changes with live-server.
  init : create a skeleton uhc app.

uhc <command>

stuck? read the docs!
  https://github.com/AyushmanTripathy/uhc#documentation

plz create an issue to report any bugs
or recommend us features we should add.
  https://github.com/AyushmanTripathy/uhc/issues
